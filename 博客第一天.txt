  🖼️ 网站所有图片更换指南

  你的博客图片分为几类，每个都有对应的存放位置和配置方法：

  1. 头像

  ┌──────────┬─────────────────────────────────────┐
  │   项目   │                详情                 │
  ├──────────┼─────────────────────────────────────┤
  │ 文件位置 │ src/assets/images/avatar.avif       │
  ├──────────┼─────────────────────────────────────┤
  │ 配置位置 │ src/config/profileConfig.ts 第 9 行 │
  └──────────┴─────────────────────────────────────┘

  更换方法：用你自己的头像图片替换 avatar.avif，支持 .avif .webp .png
  .jpg。文件名可以任意，只需在配置中修改路径：

  avatar: "assets/images/我的头像.avif",  // 改这里的文件名

  2. 网站 Logo（导航栏图标）

  ┌──────────┬────────────────────────────────────────┐
  │   项目   │                  详情                  │
  ├──────────┼────────────────────────────────────────┤
  │ 文件位置 │ src/assets/images/firefly.png          │
  ├──────────┼────────────────────────────────────────┤
  │ 配置位置 │ src/config/siteConfig.ts → navbar.logo │
  └──────────┴────────────────────────────────────────┘

  3. 桌面/移动端壁纸

  ┌──────────┬─────────────────────────────────────────────────────────┐
  │   项目   │                          详情                           │
  ├──────────┼─────────────────────────────────────────────────────────┤
  │ 文件位置 │ src/assets/images/DesktopWallpaper/d1~d6.avif （桌面）  │
  ├──────────┼─────────────────────────────────────────────────────────┤
  │          │ src/assets/images/MobileWallpaper/m1~m6.avif （移动端） │
  ├──────────┼─────────────────────────────────────────────────────────┤
  │ 配置位置 │ src/config/backgroundWallpaper.ts                       │
  └──────────┴─────────────────────────────────────────────────────────┘

  更换方法：把你的壁纸图片放进对应文件夹，然后在配置里改文件名即可。也可以使用在
  线随机图 API：

  src: {
    desktop: "https://你的随机图API地址",   // 或 ["d1.avif", "d2.avif"]
    mobile: "https://你的随机图API地址",
  }

  4. 文章默认封面图

  ┌──────────┬────────────────────────────────┐
  │   项目   │              详情              │
  ├──────────┼────────────────────────────────┤
  │ 文件位置 │ src/assets/images/cover.avif   │
  ├──────────┼────────────────────────────────┤
  │ 配置位置 │ src/config/coverImageConfig.ts │
  └──────────┴────────────────────────────────┘

  5. 赞助收款码

  ┌──────────┬─────────────────────────────────────────┐
  │   项目   │                  详情                   │
  ├──────────┼─────────────────────────────────────────┤
  │ 文件位置 │ public/assets/images/sponsor/alipay.png │
  ├──────────┼─────────────────────────────────────────┤
  │          │ public/assets/images/sponsor/wechat.png │
  ├──────────┼─────────────────────────────────────────┤
  │ 配置位置 │ src/config/sponsorConfig.ts             │
  └──────────┴─────────────────────────────────────────┘

  ▎ ⚠️ 赞助图片放在 public/ 目录（不是 src/）下，因为 Astro 
  ▎ 不会优化这些图片，保持原样。

  6. 文章内的图片

  写文章时，图片放在和文章同级的 images/ 文件夹：

  src/content/posts/
  ├── 我的文章.md
  └── images/
      └── 截图.png

  然后在 Markdown 中引用：
  ![描述](./images/截图.png)

  ┌──────────┬─────────────────────────────┐
  │   项目   │            详情             │
  ├──────────┼─────────────────────────────┤
  │ 文件位置 │ public/gallery/             │
  ├──────────┼─────────────────────────────┤
  │ 配置位置 │ src/config/galleryConfig.ts │
  └──────────┴─────────────────────────────┘

  5. Cloudflare Pages 自动部署 → 博客更新 ✨
